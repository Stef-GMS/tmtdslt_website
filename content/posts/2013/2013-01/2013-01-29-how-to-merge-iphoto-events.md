I sometimes write tutorials or How-To documents for people. I was recently asked, “_How do I merge iPhoto events?_” and I thought I would share with my readers.

[Dixie Allan](http://webclipart.about.com/bio/Dixie-Allan-97321.htm) over at [WebClipart.about.com](http://WebClipart.about.com) creates clipart that her [Terms of Use](http://webclipart.about.com/od/Categories/a/Terms-Of-Use-For-Original-Clip-Art-Created-By-Dixie-Allan.htm) allows for personal not for profit use. I use them for personal cards and things; on occasion I will use them on my blog, of course giving her credit.

I’ve just imported a bunch of her [Valentine’s Day](http://webclipart.about.com/od/holida1/ss/Clip-Art-For-Valentines-Day.htm) clipart into my “Graphics” iPhoto library so it is the perfect time to write this tutorial.

You’ll notice that when you import individual images into iPhoto it automatically creates a separate event for each image, each noted here by “untitled event”  
![wpid-iPhotoIndivEvents-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-1.jpg)

In the above, you can see in the left navigation area iPhoto shows “Photos library” is selected. You could just drag each photo into the event you want, but if you have a lot that can be tedious if you have a bunch and if they’re not right next to each other.

So, let’s work with events by click the “Event” library in the left margin and you’ll see something similar to this  
![wpid-UntitledEvents-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-2.jpg)

Now, single click the first event you wish to merge. Now while holding down the Command button click each of the other events. You’ll notice each event you select will then be outlined like this.  
![wpid-SelectEvents-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-3.jpg)

If you happen to select an event you don’t want to merge, as shown here _(Autumn is selected and well Valentine’s Day isn’t in Autumn so I don’t want it to be merged)_  
![wpid-OoopsTooManySelected-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-4.jpg)

While holding down the Command button click the event you don’t want and it will be deselected.  
![wpid-DeSelected-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-5.jpg)

Now that you have the correct events selected you have a few choices on how to Merge Events.

1) You can right-click on one of the events and choose “Merge Events”  
![wpid-MergeEventsViaRightClick-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-6.jpg)

2) Or you can go to the Events menu and choose “Create Event” or “Merge Events”  
![wpid-CreateEventMenu-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-7.jpg)

3) Or you can deselect one of the events and drag the others onto the unselected event  
![wpid-ScreenShot2013-01-29at3.28.12PM-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-8.jpg)  
![wpid-MergeEventsDragging-1-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-9.jpg)

Whatever way you choose, when the events have been merged you will see your single event displayed.  
![wpid-NameEvent-1-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-10.jpg)

If as in my case the event is untitled, you can create a name by clicking on the “untitled event” title and enter a title  
![wpid-NameEvent2-2013-01-29-17-55.png](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-11.png)

If you have sorting by name turned on (under View menu) then your even will change location. In my case it went to the bottom. I then noticed something, I now have two “Valentine’s Day” events (save for the Vintage one which I want to keep separate)  
![wpid-EventsToMerge-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-12.jpg)

So time for more merging, this time I just drug newest one on top of original  
![wpid-MergeEventsDragging-2-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-13.jpg)

There we go… now I have everything setup the way I want it.  
![wpid-EventsMerged-2013-01-29-17-55.jpg](/images/2013/2013-01/2013-01-29-how-to-merge-iphoto-events-img-14.jpg)

I hope this tutorial (How-To) was helpful. Was it?
